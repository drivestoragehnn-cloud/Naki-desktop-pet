# Naki 🐾

**Bring Your Desktop to Life.**

Naki is a lightweight desktop pet inspired by Neko98. Living directly on your desktop, Naki adds personality, fun, and interactivity to your daily computer experience.

## ✨ Features

* 🐱 Inspired by the classic Neko98 desktop pet
* ⚡ Lightweight and fast
* 🎨 Customizable
* 🖥️ Lives directly on your desktop
* 🔓 Open Source
* 💾 No installation required

## 📥 Download

Download the latest Windows executable from:

* [GitHub Releases](https://github.com/drivestoragehnn-cloud/Naki-desktop-pet/releases)
* [itch.io](https://nikan48.itch.io/naki-desktop-pet)

Simply download the `.exe` file and run it. No installation, no dependencies — just download and double-click.

## 🛠️ Built With

* Python
* PyQt6

The full source code is included in this repository for transparency. The `.exe` release is a self-contained build, so Python does not need to be installed to run Naki.
